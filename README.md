# Sulov Mitra | Portfolio Website

A modern, single-page personal portfolio website with glassmorphism design, interactive animations, and multiple theme options.

**Live:** [sulov.online](https://sulov.online)

## Tech Stack

- **HTML5** — Semantic single-page structure
- **CSS3** — Glassmorphism effects, CSS variables, grid/flexbox layouts, custom animations
- **JavaScript** — Canvas particle system, typed text effect, scroll reveal, theme switcher, form handling
- **Fonts** — [Poppins](https://fonts.google.com/specimen/Poppins) & [Fira Code](https://fonts.google.com/specimen/Fira+Code) (Google Fonts)
- **Icons** — [Font Awesome 6.5](https://fontawesome.com/)
- **Contact Form** — [Web3Forms](https://web3forms.com/) integration

## Features

- **Particle Background** — Interactive canvas-based particles that connect on proximity and respond to mouse movement
- **12 Themes** — Dark (default), Light, Navy, Purple, Green, Slate, Bumblebee, Cupcake, Retro, Emerald, Corporate, Cyberpunk
- **Scroll Reveal Animations** — Elements fade in as they enter the viewport using IntersectionObserver
- **Typed Text Effect** — Rotating role titles in the hero section
- **Responsive Design** — Mobile-friendly with hamburger menu and adaptive layouts
- **Glassmorphism UI** — Backdrop blur, transparent cards, and glow effects throughout

## Getting Started

Just open `index.html` in a browser — no build tools or dependencies needed.

```bash
git clone https://github.com/Sulovb2/My-Portfolio.git
cd My-Portfolio
# open index.html in your browser
```
